# Pong-Dong

shitty pong clone made with C++ and raylib

## how to run
### Linux/Mac OS
-- install g++
```
mkdir build
make linux
./build/linux/build.out
```

### Windows
```
mkdir build
make windows
.\build\windows\build.exe
```

go to [Pong-Dong VS2020](https://github.com/magmagaming001/Pong-Dong-VS2020)
